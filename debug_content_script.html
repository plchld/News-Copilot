Debug Content Script Issues

Steps to debug:

  1. Open Chrome DevTools on the pronews.gr article page
  2. Go to the Console tab
  3. Check if you see: "News Copilot - Ελληνική έκδοση φορτώθηκε."
  4. If not, the content script isn't loading
  5. Check for any errors in the console
  6. Run this in console: document.getElementById('augment-article-button')
  7. If it returns null, the button wasn't created
  8. Check if the button exists but is hidden: document.querySelector('#augment-article-button').style

Common issues:

  • Content Security Policy blocking the script
  • The page using iframes
  • JavaScript errors preventing execution
  • The extension needs to be reloaded after changes

To reload the extension:

  1. Go to chrome://extensions/
  2. Find "Article Augmentor"
  3. Click the refresh icon
  4. Reload the pronews.gr page
